# 泡泡爪宠物洗护

宠物洗护服务品牌 landing page，基于 Next.js App Router 构建的单页面静态站点。

## 功能

- 服务介绍（精细沐浴、毛发养护、美容造型、接送服务）
- 套餐定价展示
- 护理流程说明（4 步）
- 在线预约表单（含手机号客户端校验）

## 技术栈

- Next.js 16 + TypeScript
- React 19（App Router）
- lucide-react 图标
- 自定义 CSS（CSS 自定义属性，无 Tailwind）
- 纯静态，无后端

## 本地开发

```bash
npm install
npm run dev        # http://localhost:3000
npm run build      # 生产构建
npm run lint       # ESLint
```
